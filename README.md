# i3-video-wallpaper

requires xwinwrap-git from AUR, xdotool, mpv

support pausing the video when a window other than the desktop is focused

## Basic Usage
```bash
./setup.sh [-a] [-p video.mp4] [-h]
```
## Parameters
```
$ ./setup.sh -h
Options:
        -a: Always run video wallpaper.
        -p: Path to video.
        -h: Display this text.
```
